# ashish.dev

Personal portfolio showcasing my work as a Frontend Engineer.

## Tech Stack

- Next.js 14 with App Router
- TypeScript
- SCSS Modules
- React 19

## Getting Started

```bash
npm install
npm run dev
```

## Development

```bash
npm run build    # Build for production
npm run start    # Start production server
npm run lint     # Run ESLint
```
